Install WorkDay on iPhone

WorkDay is a web app that installs to your home screen and uses the same admin login and live data as the website.

  1. Open this page on your iPhone using Safari (not Chrome).
  2. Tap the button below to open the WorkDay app.
  3. Tap the Share button (□↑) at the bottom of Safari.
  4. Scroll and tap Add to Home Screen.
  5. Confirm the name WorkDay and tap Add.

Changes on the phone app and the website admin sync automatically because both use the same cloud database.
